File prefix: origin.

* `oj*`: Online Judge. To see the problem descriptions, go to [http://uva.onlinejudge.org](http://uva.onlinejudge.org), or google "uva " + problem number.
* `gcj*`: Google Code Jam.
* `var*`: various.

File extension: language.

* `*.java`: Java (1.6+).
* `*.c`: C.
* `*.rb`: Ruby (1.9+).
* `*.hs`: Haskell.
